WHAT IS THIS WEBSITE?

The website https://www.loosingbraincells.ml/ was originally just a website to place HTML, CSS, and JavaScript code for fun, but after people wanted to know how I made those things, I decided to start this website with a friend to teach people how to make cool web-based projects.

<HTML>

HTML is the most popular and most simple web programming language. With HTML, you can make some static pages. With the help of CSS, you can style it and make it look nice. With JavaScript, you can make it interactive. There is so much more you can do, too. Even this page is made with HTML and CSS!

SCRATCH

Scratch is a block-based programming language developed at MIT University. With Scratch 3.0 released, you can embed your own games, calculators, and more into HTML5. Visit my partner's website at https://zund.cf where he showcases his own Scratch Projects.
